# FFmpeg 工具箱

基于 FFmpeg 的图形化音视频处理工具，提供简洁美观的界面，支持多种音视频操作。

## 功能特性

- **视频拼接** - 将多个视频文件合并为一个
- **音频提取** - 从视频文件中提取音频
- **音频转码** - 转换音频格式，支持码率设置
- **音视频合成** - 将视频和音频合并
- **视频转码** - 转换视频格式，支持分辨率和码率调整
- **图片转视频** - 将图片序列转换为视频

## 界面预览

- 现代化卡片式UI设计
- 左右分栏布局（功能区 + 日志区）
- 彩色状态指示器
- 实时日志输出

## 使用要求

1. Python 3.x
2. FFmpeg 可执行文件

## 安装 FFmpeg

### Windows
1. 访问 https://ffmpeg.org/download.html
2. 下载 Windows 版本并解压
3. 记录 `ffmpeg.exe` 的完整路径

### Mac
```bash
brew install ffmpeg
```

### Linux
```bash
sudo apt install ffmpeg
```

## 使用方法

### 1. 配置 FFmpeg 路径

打开 `ffmpeg_tools_gui.py`，修改第 19 行的路径：

```python
self.ffmpeg_path = r"D:\Apps\ffmpeg\bin\ffmpeg.exe"  # Windows 示例
# 或
self.ffmpeg_path = "/usr/bin/ffmpeg"  # Mac/Linux 示例
```

### 2. 运行程序

```bash
python ffmpeg_tools_gui.py
```

### 3. 使用功能

- 选择对应功能的标签页
- 选择输入文件
- 设置参数（如需要）
- 点击执行按钮
- 查看右侧日志输出

## 各功能说明

### 视频拼接
1. 点击 "+ 添加视频" 添加多个视频
2. 使用 "↑ 下移" 调整顺序
3. 选择输出文件路径
4. 点击 "▶ 开始拼接"

### 音频提取
1. 选择输入视频文件
2. 选择输出格式（mp3/aac/wav/flac/ogg/m4a）
3. 选择输出文件路径
4. 点击 "▶ 提取音频"

### 音频转码
1. 选择输入音频文件
2. 选择输出格式和码率
3. 选择输出文件路径
4. 点击 "▶ 开始转码"

### 音视频合成
1. 选择视频文件
2. 选择音频文件
3. 选择输出文件路径
4. 点击 "▶ 开始合成"

### 视频转码
1. 选择输入视频
2. 选择输出格式、分辨率、码率
3. 选择输出文件路径
4. 点击 "▶ 开始转码"

### 图片转视频
1. 选择包含图片的文件夹（图片需按数字命名：1.jpg, 2.jpg...）
2. 设置帧率和图片格式
3. 选择输出视频路径
4. 点击 "▶ 开始生成"

## 界面配色

- 主色调：蓝色 `#2196F3`
- 成功：绿色 `#4CAF50`
- 错误：红色 `#f44336`
- 背景：浅灰 `#f5f5f5`

## 日志说明

- **绿色** - 成功信息
- **红色** - 错误信息
- **蓝色** - 普通信息

## 注意事项

1. 确保 FFmpeg 路径配置正确
2. 处理大文件时需要耐心等待
3. 输出文件路径不要包含特殊字符
4. 图片转视频时图片需按数字顺序命名

## 技术栈

- Python 3.x
- tkinter - GUI 框架
- FFmpeg - 音视频处理
